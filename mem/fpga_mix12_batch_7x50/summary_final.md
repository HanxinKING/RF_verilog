## 结论

- 当前 RTL 与 Python 定点结果完全一致，说明 FPGA/RTL 实现已经准确复现了定点参考链路。
- 当前 RTL 与 Python 浮点结果一致率为 `0.957143`，说明主要差异来自浮点到定点的量化误差，而不是 RTL 实现错误。
- 从真实标签角度看，类别 `1`、`2`、`5` 识别效果最好，类别 `3`、`4`、`6` 仍然是主要薄弱类别。
